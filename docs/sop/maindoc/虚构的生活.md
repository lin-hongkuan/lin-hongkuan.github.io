# 虚构的生活
很早之前有位刚认识的朋友和我说，他很羡慕我的生活。他想过我的生活。

我下意识表现得很诧异。因为身边的朋友都知道我的生活其实过得很糟糕。但很快又为他而感到由衷的开心。

我向他解释，他对我的生活的观察和理解其实和我早就无关了。他向我描述的那些对我生活的形容并不是真的，这其实是他所向往的生活的方式。他虚构了我的生活，并从这种虚构中捕获了能够支撑他生活下去的力量。而那些真实的部分已经交由虚构“润化”，从“我的手里”悄悄转变为“他的口中”之物。尽管只是一种描述，但这个生活早已不属于我。

但我并不介意这些。我很感谢他的慷慨。他的这些话令我突然发觉自已曾经的一些愚蠢的偏见。为何总是要期盼从现实当中汲取信心？为何要竭力地尝试证明自己的生活是真实的？对虚构的偏见从来不会佐证我们有多么热爱我们的真实，相反，我第一次认识到这种行为可能正是对真实的一次偷窃。用滑稽的努力去规划真实和虚构在我们生活中的比例，这种愚蠢忽视了故事的善良和善良的故事的可能性。

如果说人们能够由虚构的组织内寻觅继续生活下去的勇气，那么这正是它的慷慨。或许说，对我的生活的描述从来都不曾属于我。有一种吝啬且总是令人尴尬广告语会欺骗我们：谁也无法定义我们的生活！但这对我来说只是一种假惺惺的愤怒。

故事是这样。故事已经发生。从虚构之中捕获恰当的能量几乎证明了，讲故事的人的慷慨，和真实被改造之前的空白的平坦。真实“平行”地走在路上，而虚构则适时地作出相应的“垂直”。慢慢的一种缤纷且强烈的力量也许就滋生在这样“讲述”的凹陷当中。

再一次感谢他的慷慨。我肯定也会有我想过的虚构的生活。
