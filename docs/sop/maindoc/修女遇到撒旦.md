# 修女遇到撒旦

## Indika
![alt text](/picture/image.png)
修道院。巨大的石块，被冰雪覆盖。世界变成了灰白与深蓝的单调。Indika（下面称呼其为伊蒂卡）独自一人，她的身体轮廓在雪地上被拉成一道黑色的剪影，与背后宏大的建筑形成了残酷的对比。她仿佛被抛弃在世界边缘。她凝视着那座带有十字架的塔楼——一座对神圣权力永恒的纪念碑。

这是独立游戏indika游戏开篇不久的一幅画面。《INDIKA》是由 Odd Meter 开发、11 bit studios 发行的一款第三人称剧情驱动型冒险游戏。它将 19 世纪末俄罗斯的宗教压抑与超现实的荒诞主义融为一炉。在这个游戏中，玩家扮演伊蒂卡————一位正在适应修道院生活的修女。故事的关键点在于她跟恶魔撒旦的联系，或者说是关系。

《INDIKA》通过伊蒂卡的一段旅途（与她脑海中的恶魔同行），探讨了一系列关于信仰，存在焦虑和自我认同的命题。有人说艾迪芬奇的记忆（《What Remains of Edith Finch》）是马尔克斯游戏化，那么《INDIKA》应该算是陀思妥耶夫斯基的游戏化。

从叙事与玩法的结合来看，《INDIKA》并不是一个依靠高互动性驱动的游戏；相反，它更接近一种**“玩家必须体验其限制”的文本。限制本身就是叙事实体。线性关卡、有限的互动、偶尔笨拙的跳跃，都强化了一种“神并未介入、世界仍然沉重”的叙事态度。这种玩法哲学使《INDIKA》更接近可操控的文学作品（playable literature）**。

## Absurd 荒诞主义的体验
### 荒诞的幽默与关卡设计
重复中的荒诞：井边的水与永恒的劳作

（此处插入井边打水的截图）

游戏最先呈现的不是奇观，而是繁琐：
伊蒂卡被要求去井边打水，而你必须忠实地替她完成——一次、两次、三次……
次数多得像是有人故意隐藏了“为何如此”的答案，而你只能在沉默的雪地里机械重复。

这一段常被玩家戏称为“打工模拟器”，
但它的本质远比滑稽更深一层：它是宗教仪式的浓缩，是修道生活的隐喻，是“意义死去后动作仍继续”的见证。

荒诞不是从夸张开始，而是从 意义与行为的脱节 开始的。
